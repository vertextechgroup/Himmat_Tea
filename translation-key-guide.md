## Usage Instructions

To use these translations in your components:

1. Import the `useTranslation` hook:
   ```tsx
   import { useTranslation } from '../../../hooks/useTranslation';
   ```

2. Use the hook in your component:
   ```tsx
   const { t } = useTranslation();
   ```

3. Replace hardcoded text with translation keys:
   ```tsx
   <h1>{t('dashboard.settings.title')}</h1>
   ```

## File Locations

- English translations: `src/locales/en.json`
- Nepali translations: `src/locales/ne.json`
- Hindi translations: `src/locales/hi.json`
- Chinese translations: `src/locales/zh.json`
- Japanese translations: `src/locales/ja.json`

## Notes

- Always use the translation key pattern: `dashboard.[component].[key]`
- Keep keys descriptive and consistent
- Update this guide when adding new translation keys
- Test translations in all supported languages
